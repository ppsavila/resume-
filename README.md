# ppsavila · Portfólio

Site de portfólio e currículo online de **Pedro Paulo S. Avila** — Tech Artist, UI Engineer e Game Developer.

## Sobre o site

Uma single-page application que apresenta minha trajetória profissional de forma visual e interativa, com seções de:

- **Hero** — apresentação e tagline
- **Sobre** — resumo da minha carreira
- **Skills** — habilidades técnicas e artísticas
- **Experiência** — histórico profissional (Fanatee, Tralingo, etc.)
- **Projetos** — projetos pessoais e profissionais selecionados
- **Contato** — links e formas de contato

## Stack

- **React 19** + **Vite**
- **Framer Motion** — animações
- **React Icons** — ícones
- **React Scroll** — navegação suave entre seções
- CSS customizado com tema escuro e acentos em verde

## Rodando localmente

```bash
npm install
npm run dev
```
